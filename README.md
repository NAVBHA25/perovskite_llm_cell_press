# Perovskite analysis using LLMs/ML
Prediction and inverse engineering of perovskite solar cells using open-access datasets, using a combination of large language models and classical machine learning. 
This paper analyzes existing, publicly available data, accessible at DOI: \url{https://doi.org/10.1038/s41467-022-35400-4} and DOI: \url{https://doi.org/10.1038/s41560-021-00941-3}. 
All the codes are structured to provide support to the analysis carried out in the main paper and additional associated information. Any additional information required to reanalyze the data reported in this paper is available from the lead contact upon request.
